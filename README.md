# Frontend Template Next.js

This is a frontend template with Next.js and TypeScript.

## Features

- [x] Next.js
- [x] TypeScript
- [x] Tailwind CSS
- [x] ESLint
- [x] Prettier
- [x] Husky
- [x] Lint-staged
- [x] React Compiler
- [x] ESLint React Compiler
- [x] i18n-js
- [x] Zod (Validation)
- [x] Proprietary backend API handling
- [x] React Hook Form (Form handling)
- [ ] Toast
- [ ] Zustand (State Management)
- [ ] Supabase (Auth and Database) handling
